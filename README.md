# Defensive Formation Playbook 🛡️

A comprehensive reference document covering the origins, evolution, tactical principles, and mapped battle-site locations of classic defensive formations — from the Greek phalanx to modern trench warfare.

## What's Inside

| Formation | Era | Key Battles |
|---|---|---|
| **Greek Phalanx** | 7th–2nd c. BC | Thermopylae, Marathon, Plataea, Leuctra |
| **Shield Wall** | Antiquity–11th c. AD | Hastings, Ashdown, Maldon, Stamford Bridge |
| **Roman Testudo** | 3rd–2nd c. AD | Alesia, Masada, Dacian Wars, Carrhae |
| **Trench Warfare** | 19th–20th c. | Verdun, Somme, Ypres, Passchendaele |

## Getting Started

1. Open **`DEFENSIVE_FORMATIONS.md`** for the full reference — formation histories, tactical principle tables, key battle narratives, geocoded location data, and an evolution timeline.
2. Use the **JSON coordinate block** in the Location Index section to import battle sites into your mapping tool of choice (Google Maps, Leaflet, QGIS, etc.).
3. Expand the playbook: add new formations, battles, or tactical analysis by forking and editing.
